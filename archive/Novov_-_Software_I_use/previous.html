Software I use

This page has programs, web apps, and software-related stuff that I use. Most are cross-platform, easy to use, and free or cheap.

SVGOMG

Free

SVGOMG

Type: Web app

Developer: Jake Archibald

Like graphic design or vector art? This is an online GUI for the popular "SVGO" tool that allows you to optimise .SVG files.

itch.io

Free (some items paid)

itch.io

Type: Online games store

Developer: itch

More of a repository or store than an actual app, it has lots of neat little experimental games. A lot of indie creators that went on to become the Next Big Thing™ released stuff here first.

Affinity Suite

US$50 each

Affinity Suite

Type: Cross-platform applications

Developer: Serif Europe

In my opinion, Adobe's monopoly on creative applications is extremely detrimental. Serif creates one of the best alternatives, for a much cheaper price. Although their apps don't quite have the extensive feature set of the Creative Cloud, they've still pretty great.

AlternativeTo

Free

AlternativeTo

Type: Website about apps

Creator: 27 Kilobyte AB

A great tool for finding alternatives to competitors or replacements for everyday apps you use.

Twine

Free

Twine

Type: Web and desktop app

Developer: Interactive Fiction Technology Foundation

A neat little app for composing interactive fiction (think Choose Your Own Adventure books that you read as a kid). For the more programatically-inclined, the look and functionality can be improved using JavaScript.

Every Time Zone

Free

Every Time Zone

Type: Web app

Developer: Slash7 LLC

The best way of working out time zones. Used to be littered with less intrusive popups, but it's still pretty good.

Analyze My Writing

Free

Analyze My Writing

Type: Web app

Developer: ???

Work out how advanced your writing is. Are you worse than you think you are? (Oh, by the way, this description is apparently at a third grade level in terms of complexity. Ouch.)

MagicaVoxel

Free

MagicaVoxel

Type: Cross-platform application

Developer: ephtracy

The most commonly used voxel model editor - i.e. based on three-dimensional cubes. Similar, but distinct, is SpriteStack, which is instead based on stacked two-dimensional pixel art, and SpryTile and CrocoTile, which instead use pixel art as a basis to build proper 3D models.

Interesting unheard of developer tools

Type: Reddit post

Creator: Multiple

A bunch of neat tools for game development - obviously, I haven't used all of them, but I can attest that some are good.

CyberChef

Free

CyberChef

Type: Web app

Developer: Open-source

A secure tool to do a bunch of common data-related programming tasks.

FontDrop

Free

FontDrop

Type: Web app

Developer: Viktor and Clemens Nübel

Upload a font and check its OpenType features, language support, specimens, and other metadata. Then compare it with other font files. Works offline once the page is loaded.

Doodad.dev

Free

Doodad.dev

Type: Web app

Developer: Nathaniel

A website with tools for generating creative and efficient web graphical elements. Currently includes an image dithering tool and a pattern generator.
